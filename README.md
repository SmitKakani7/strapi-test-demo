### `develop`

```
npm run develop
# or
yarn develop
```

### `start`

```
npm run start
# or
yarn start
```

### `build`

```
npm run build
# or
yarn build
```

## Check locally

- clone repository
- start the project
- hit this request in postman
```
            url: http://localhost:1337/api/posts
            method: POST
            raw-JSON: {
                "data": {
                    "title": "Please enter title here"
                }
            }
```
- check the strapi admin for result
